Spotify Web Browser

A social media application that helps users connect virtually sharing their lives with others through posts, pictures, videos and more! Users can sign up and log into their own personal accounts following other users on the platform. Every user has a live feed of posts from all the users they follow.

Features
  • User authentication for signing up with an email and password and logging in.
  • Profile page to check account info and user related items such as user feed and followers.
  • Home post feed page to view all live posts from followed users.
Website Technology
  • Front End: React, Material UI
  • Back End: MongoDB, Mongoose
  • Frameworks/Tools: Next.js, Express.js, Passport.js
