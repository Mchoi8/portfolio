Spotify Web Browser

Description

A clone of the Spotify Web Player with a search feature that looks up artists, albums, and tracks. Users are shown detailed information on their results such as related songs or artists or top 10 tracks from an artist.

Features

  • Main page with the user's profile and search bar to look up spotify's info.
  • A page dedicated to a artist, album, or song giving detailed information such as the specified genre and ratings.

Website Technology

  • Front End: Angular
  • Back End: Node.js, Express
  • Frameworks/Tools: Spotify API, Spotify Developer Dashboard
