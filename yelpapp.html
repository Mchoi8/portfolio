Projects


I'm Feeling Hungry

Description

A mobile application featuring a Yelp clone to help users decide on a place to eat by giving them a quiz that identifies their dining preferences.

Features

  • Used firebase for user authentication and storing user preferences and personal data.
  • Integrated a quiz with multiple choice questions for users to fill out.
  • Utilized the Yelp API to retrieve data from restaurants.

Website Technology

  • Languages and Tools: Java, Firebase, Yelp API, Android Retrofit
  • Other technologies: Android Studio
